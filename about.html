About Smart Campus Lost & Found

The Smart Campus Lost & Found System is a web application designed to help students and staff report, search, and recover lost items on campus.


Project Objectives

  • Report lost items quickly.
  • Report found items easily.
  • Search for lost and found items.
  • Improve item recovery on campus.
  • Provide a user-friendly digital solution.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Progressive Web App (PWA)
  • Local Storage
  • GitHub Pages
